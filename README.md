# tab-bar
   底部导航栏,可自定义导航项
   点击后icon变为icon-active同时跳转路由
   props: link 点击后跳转到对应的路由
   tab-bar-item:tab-bar的子组件,拥有 item-icon, item-icon-active, item-text 三个插槽
## Project setup
```
npm install
```
